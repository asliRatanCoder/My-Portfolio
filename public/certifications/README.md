# Certification Images

Place your certification images in this folder:

- `aws.png` - AWS Certified Solutions Architect
- `oracle.png` - Oracle Certified Professional, Java SE
- `react.png` - React Developer Certification
- `scrum.png` - Certified Scrum Master (CSM)

Recommended image size: 200x200 pixels or larger (square format)
Supported formats: PNG, JPG, SVG
