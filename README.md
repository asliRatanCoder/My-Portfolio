# Portfolio Website

A modern, responsive portfolio website built with React and Vite, featuring a clean blue and white color theme.

## Features

- **About Me** - Personal introduction and technical skills
- **Work Experience** - Professional work history with achievements
- **Education** - Academic background and qualifications
- **Certifications** - Professional certifications and credentials
- **Responsive Design** - Mobile-friendly layout
- **Modern UI** - Blue and white color scheme with smooth animations

## Tech Stack

- **Frontend**: React 18
- **Build Tool**: Vite
- **Styling**: CSS3 with custom properties
- **Future Backend**: Java (for API endpoints)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build

## Customization

To customize the portfolio with your own information:

1. Update the content in each component located in `src/components/`
2. Modify the color scheme in `src/index.css` CSS variables
3. Add your own images and links
